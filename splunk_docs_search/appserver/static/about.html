Splunk Docs Search

Browse Splunk product documentation offline — faithful HTML topics with a version picker, search, and dark mode — inside Splunk.

Views

  • Documentation — the offline docs browser (product tabs, version dropdown, search, dark mode).
  • SPL Search — full-text search over the indexed topics for power users.
  • Configuration — download or update docs from help.splunk.com and watch progress.

How docs get here

The Configuration page runs scraper/fetch_docs.py on the Splunk host, which fetches topics from help.splunk.com into appserver/static/docdata/ (HTML topics, images, nav, and a search index) and an NDJSON feed for the splunk_docs index. You can also run the scraper standalone on a staging host and copy the docdata/ folder in — ideal for air-gapped installs.

Coverage and how many versions to keep are set in scraper/products.yaml.
